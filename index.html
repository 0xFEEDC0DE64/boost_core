Boost.Core

The Boost.Core library is a collection of core utilities. The criteria for inclusion is that the utility component be

  • simple,
  • used by other Boost libraries, and
  • not dependent on any other Boost modules except Core itself, Config, Assert, Static Assert, or Predef.

Currently, the Core library contains

  • boost::addressof
  • boost::checked_delete
  • boost::enable_if
  • boost::noncopyable
  • boost::ref
  • boost::swap
  • boost::null_deleter
  • BOOST_EXPLICIT_OPERATOR_BOOL, BOOST_EXPLICIT_OPERATOR_BOOL_NOEXCEPT and BOOST_CONSTEXPR_EXPLICIT_OPERATOR_BOOL
